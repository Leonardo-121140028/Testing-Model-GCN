# Testing-Model-GCN

### Saat ini alur program yaitu 
- mengekstrak video menjadi frame menggunakan  video_frame_extract.py dengan memanfaatkan file datatest_desc.xlsx untuk mengetahui rentang video mana yang harus diekstrak
- mengekstrak data skeleton dari frame menggunakan extract_skeleton.py
- menjalankan prediksi data skeleton menggunakan prediction_terbaru.py, menghasilkan file predictions_all.csv dan predictions_all_confusion_matrix.png
